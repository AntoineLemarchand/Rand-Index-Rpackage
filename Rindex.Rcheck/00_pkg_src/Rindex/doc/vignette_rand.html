Indice de Rand

This vignette will be about Rand index and its utilization. For that, we will explain how it works with practical examples. Here, we will change 4 variables : Mu, Sigma, n.norms and the number of observations. In this vignette like the Jaccard one, we will keep the same methods to be able to see differences between each index.

In this vignette, “single” and “average” will be our tested methods.

Reference

This example will be a reference for the 4 examples that will follow. With this, we will be able to see the variation of each of our variables.

n = 4
Mu = 5 
sigm = 2       
nb_obs= 1000

df <- clustering(dataframe = norme(n.norms = n,n = nb_obs,mu = Mu,sigma = sigm),
                 k = 3)           #creation of the data frame and implementing it 
                                  #in our function clustering


rand(dataframe = df,
        method1 = "single",
        method2 = "average")    #choice of method for our rand function 
#> [1] 0.955404

ggplot(df) +                    
  aes(x,y) + 
geom_point(aes(color=average))  #creation of the graphic
